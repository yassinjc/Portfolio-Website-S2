dataoverzicht

Aanpak

Als het gaat om development, heb ik nog heel weinig ervaring met Javascript. En met heel weinig bedoel ik bijna geen, dus om een gevoel te krijgen voor Javascript zocht ik naar wat beginners projectjes om te volgen online. Hiermee kon ik een gevoel krijgen voor Javascript en kon ik de kennis die ik daarvan opnam meenemen in het maken van mijn portfolio. Ik kwam uit op deze tutorial waar 3 projecten uitgelegd werden om te maken. Met deze GitLab link kan je het tweede project, genaamd "Dataoverzicht", bekijken die ik heb gemaakt. Het doel was om data in overzicht te brengen door middel van Javascript. Dit project ging bij mij als volgt.


Het proces

Ik begon met een simpele HTML layout waarin ik CSS en Javascript aan toevoegde. Het enige wat ik daarna heb toegevoegd was een kopje en een class voor alle data genaamd "info".
Vervolgens heb ik minimale CSS toegevoegd om alles een beetje netjes weer te geven.
Vervolgens begon ik aan het toevoegen van Javascript. Ik gebruikte "let" om meerdere variabelen op te slaan op 1 plek met een array, ik zette mijzelf en mijn vrienden hierin met iedereen's leeftijd. Dat was het kleine stukje aan data wat ik wilde weergeven. Vervolgens maakte ik een variabel "info" die de "info" class selecteerde. Vervolgens gebruikte ik opnieuw "let" om een functie aan te roepen. Binnenin deze functie zorgde ik ervoor dat alle gegevens binnenin de class, die ik gemaakt had in HTML, gezet werden door middel van een return statement. In de return statement zette ik de naam en leeftijd, als items, in de class en om het wat completer te maken voegde ik ook stukjes tekst toe om hele zinnen te maken van de datastukjes. Onderin zie je ook een stukje code "info.innerHTML = details.join('\n');", hiermee kon ik de data daadwerkelijk weergeven. Hieronder zie je het eindresultaat.



Reflectie

Deze opdracht was ietsjes moeilijker te volgen in het begin, maar uiteindelijk best goed te begrijpen. Hiermee heb ik ook een beetje geleerd hoe ik data kan weergeven met javascript. Wel kleinschalig maar begrijpelijk en handig om te kunnen.
