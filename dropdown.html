dropdown menu

Aanpak

Ik kreeg van Jo-An en Lisette de feedback om de navigatie binnenin mijn portfolio te verbeteren. Het was makkelijk om verdwaald te raken in mijn portfolio en dat is uiteraard niet gebruikersvriendelijk, dus ik kwam met het idee om een dropdown menu te maken waarin alle leerdoelen heel makkelijk bereikbaar zijn ongeacht waar je je bevindt in mijn portfolio. Via deze GitLab link kan je de code bekijken van het menu. Dit is het proces van het maken van dat menu.


Het proces

Om te beginnen met het maken van de dropdown menu heb ik een button class gemaakt waarmee je het menu in beeld krijgt als je erover hovered. Vervolgens heb ik daar een unordered list class gezet met de een knop om naar alle leerdoelen te gaan, zowel als alle leerdoelen individueel. Hiermee wordt het navigeren een stuk makkelijker.
Vervolgens begon ik aan de CSS. Ik stijlde het menu op een manier wat aansloot op de rest van mijn website. Ik had alle knoppen onder elkaar gezet, de tekst een oranje kleur en gloei gegeven, en de kleur van de knop zelf ook donker oranje gemaakt om uiteindelijk een mooi resultaat te krijgen. Vervolgens riep ik de dropdown menu zelf aan door "display: block" toe te voegen aan de unordered list class die ik had gemaakt als er gehovered werd over het leerdoelen knopje.
