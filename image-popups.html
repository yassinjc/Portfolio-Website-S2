Aanpak

In mijn POC-documentaties gebruik ik altijd wel foto's om het proces duidelijker uit te kunnen leggen. In de eerste versie van mijn portfolio voegde ik de foto's in en deed er daarna niks meer aan. Op een gegeven moment vond ik dit een beetje een levenloos en statisch gevoel geven dus ik ben direct op zoek gegaan naar manieren om dit interactive te maken. Uiteindelijk kwam ik op "Modal Image Pop-ups" en hieronder lees je hoe ik dat heb gebruikt in mijn website.


W3Schools

Ik ben op deze image techniek gekomen door de website W3Schools, hierin werd uitgelegd hoe de code hiervoor werkt en wat alle stappen waren om dit succesvol in je eigen website te zetten.
Voordat ik de stappen uitleg, hier kun je de mijn code zien voor de modal image pop-ups: GitLab
Stap 1 was HTML natuurlijk, ik had de foto's van te voren al ingevoegd en vervolgens moest ik een Modal div, toen moest ik een close button maken met een span element zodat je ook uit de pop-up kan klikken, vervolgens moest ik een image class maken waarin ik de fotos oproep en ten slotte moest ik nog een caption div maken voor eventuele tekst voor onder de foto('s).
Stap 2 was om CSS toe te voegen voor alle elementen die ik heb aangemaakt in HTML.
Stap 3, wat ook de laatste stap was, was om Javascript toe te voegen. Allereerst moest ik de modal oproepen met een variabele. Daarna moest ik een aantal variabelen maken voor de foto('s) zelf en de caption tekst, vervolgens moest ik een functie maken waarmee er ook daadwerkelijk een pop-up komt als er op de foto('s) geklikt wordt. De laatste stap was om de close button op te roepen en een functie te geven waarmee hij je uit de foto kan halen.
