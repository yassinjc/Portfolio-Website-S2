user tests | portfolio design

Aanpak

Na uitleg gekregen te hebben over hoe je User Tests op kan stellen, ben ik direct gaan denken aan een User Test voor mijn portfolio. Anke had 1 manier uitgelegd van User Tests waarin een gebruiker bepaalde woorden krijgt om hetgene te beschrijven wat ze te zien krijgen, dit leek mij een goed idee voor mijn portfolio. Voor mijn User Test heb ik de volgende woorden gebruikt om mijn portfolio design te beschrijven:
Rustgevend, druk, overzichtelijk, onoverzichtelijk, aangrijpend, nietszeggend, oppervlakkig, nieuwsgierigheid-wekkend, chaotisch, strak, onprofessioneel, duidelijk, onduidelijk, modern, ouderwets, saai, betrouwbaar en aantrekkelijk.
Ik heb 3 mensen van compleet verschillende leeftijden gevraagd om een User Test uit te voeren:
Sophie, mijn vriendin van 17 jaar oud. Mazouni, mijn vader van 53 jaar oud en Ineke, Sophie's oma van 74 jaar oud.
Ik heb hun alle 3 dezelfde selecties aan woorden gegeven en gevraagd om degene aan te vinken die overeen kwamen met wat voor uitstraling mijn portfolio had, na het gekeken- en gebruikt te hebben van mijn portfolio. Hieronder zie je de resultaten.



Het resultaat

Uit dit onderzoek bleek dat, ongeacht de leeftijdscategorie, de portfolio overzichtelijk, strak, duidelijk, modern en aantrekkelijk is. De jongere 2 leeftijdsgroepen vonden de portfolio ook nieuwsgierigheid-wekkend en betrouwbaar. Maar over het algemeen was dit hetgene waar ik voor schoot met het maken van mijn portfolio en dat blijkt uit deze User Test ook zo aan te komen bij de gebruiker. Zeker tevreden.
