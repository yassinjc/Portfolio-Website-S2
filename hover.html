hover animatie

Aanpak

Als het gaat om development, heb ik nog heel weinig ervaring met Javascript. En met heel weinig bedoel ik bijna geen, dus om een gevoel te krijgen voor Javascript zocht ik naar wat beginners projectjes om te volgen online. Hiermee kon ik een gevoel krijgen voor Javascript en kon ik de kennis die ik daarvan opnam meenemen in het maken van mijn portfolio. Ik kwam uit op deze tutorial waar 3 projecten uitgelegd werden om te maken. Met deze GitLab link kan je het derde project, genaamd "Hover", bekijken die ik heb gemaakt. Het doel was om een simpele animatie te maken met Javascript. Dit project ging bij mij als volgt.


Het proces

Wederom begon ik met het maken van een simpele layout in HTML waarin ik natuurlijk CSS en Javascript aan heb toegevoegd. Buiten dat heb ik een class gemaakt waarmee ik een cirkel weergeef, deze ben ik gaan maken in CSS.
Buiten de standaard CSS die ik toevoegde om de cirkel gecentreerd te maken, maakte ik de cirkel zelf met een blauwe kleur en een border-radius van 100px. Hiermee kreeg ik een blauwe cirkel. Ook voegde ik een cubic-bezier transition toe om een smooth animatie te krijgen in het eindresultaat. Ten slotte voegde ik ook een transform aan om de cirkel te vergroten tijdens de animatie.
Ik begon aan de Javascript met het maken van een variabel voor de cirkel die de gemaakte cirkel selecteerde. Vervolgens voegde ik een event listener toe, deze zorgde er met "mouseenter" voor dat als er over de cirkel werd gehovered, er gecheckt werd of er geen hover was. Als dat het geval was werd een juist een hover toegevoegd. Hiermee kreeg ik de animatie aan de gang, alleen bleef hij groot dus om dit te voorkomen maakte ik nog een event listener. Deze zorgde er met "mouseleave" voor dat als de muis niet meer hoverde over de cirkel, dat de hover van de cirkel eraf ging als die er uberhaupt op zat.
Uiteindelijk was dit het resultaat.



Reflectie

Deze opdracht was waarschijnlijk het makkelijkste van de 3 en dat komt vooral door de opdrachten voorheen. Ik begreep een beetje beter hoe Javascript werkte dus alles was best te doen deze keer. Ik ben zeker tevreden met de kleine progressie die ik heb gemaakt qua kennis over Javascript.
